# iot-device server

## Environment

### Dependency manager

As packaging and dependency management system use [Poetry](https://python-poetry.org/).

To force Poetry store environment inside server's directory run command:
```shell
poetry config virtualenvs.in-project true
```

### Install

```shell
poetry install
```
